## 친절한 SQL 튜닝
저자: 조시형

## 느낀 점
```text
책 뒤에 색인으로 여러 개념들이 적혀져있어서
영단어 외우듯이 모르는 개념은 다시 학습하면서 여러번 읽었던 것 같다.
이 책은 운영체제나 알고리즘 개념을 함께 끌어왔기에,
내가 바라보는 sql이 단순히 쿼리 언어가 아니라
자원 경쟁 환경에서 돌아가는 하나의 프로그램으로 인식하는 법을 알게 해주었다.
튜닝의 본질이 문법이 아니라 자원 점유 시간을 최소화하는 실행 흐름 설계라는 것을
명확히 보여주었다.
실제 sql이 느려지는 이유를 눈으로 확인하고
체험할 수 있어서 재미있었다.
```
